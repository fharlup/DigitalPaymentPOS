Soto Mbak Eni - Digital Payment & POS System
Sistem Point of Sales (POS) dan manajemen keuangan terintegrasi yang dibangun untuk mendigitalisasi operasional bisnis kuliner lokal. Proyek ini mencakup seluruh siklus pengembangan, mulai dari elisitasi kebutuhan fungsional (FR) hingga tahap hosting dan implementasi di lapangan.

Business Problem
Sebelumnya, pencatatan transaksi dan pelaporan keuangan dilakukan secara manual, yang sering kali menyebabkan ketidakkonsistenan data dan kesulitan dalam melacak pembayaran digital. Sistem ini hadir untuk mengotomatisasi proses tersebut dan mendukung metode pembayaran nontunai yang modern.

Tech Stack
Framework: Laravel Filament (untuk manajemen admin dan dashboard yang cepat).

Payment Gateway: Midtrans API (Integrasi QRIS & Bank Transfer).

Database: MySQL.

Testing: Test Cases untuk fungsionalitas inti.

Key Features
Automated POS: Manajemen pesanan dengan status pembayaran real-time (pending/paid).

QRIS Integration: Integrasi penuh dengan Midtrans, memungkinkan pelanggan membayar via QRIS dengan notifikasi otomatis ke sistem.

Analytical Dashboard: Statistik penjualan dan grafik pendapatan harian, mingguan, serta bulanan untuk membantu pengambilan keputusan bisnis.

Financial Reporting: Rekapitulasi otomatis yang meminimalisir kesalahan pencatatan manual.

Impact & Results
Efisiensi Operasional: Berhasil meningkatkan efisiensi pencatatan transaksi sebesar 40% dibandingkan metode manual.

Digitalisasi Pembayaran: Mendukung transaksi nontunai sepenuhnya, memudahkan pelanggan dan mempercepat proses verifikasi pembayaran bagi pemilik usaha.
